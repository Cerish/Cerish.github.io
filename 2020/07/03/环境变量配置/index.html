环境变量配置

Posted on 2020-07-03

1. 软件环境变量配置

1.1 配置Java环境变量

  • 右键 计算机 –> 属性 –> 高级系统设置 –> 环境变量
  • 新建系统变量 –> Java_Home 值为你下载的Java 路径 eg: G:\huajieli\jdk
  • 编辑 Path变量：%Java_Home%\bin;%Java_Home%\jre\bin; 注意变量之间的分号
  • 新建CLASSPATH变量， 值为 .;%Java_Home%\bin;%Java_Home%\lib\dt.jar;%Java_Home%\lib\tools.jar

点击确定，完成。

1.2 配置maven变量

  • 下载maven包， 解压到任意文件夹
  • 在环境变量中添加， G:\huajieli\maven\apache-maven-3.6.3\bin

1.3 配置 git 环境

  • 在环境变量中添加， G:\huajieli\Git\cmd\git.exe

  • 右键配置git

    • cmd –> regedit, 定位到 HKEY_CURRENT_USER\SOFTWARE\Classes\Directory\Background\shell
    • 新建 bash–> Icon : G:\huajieli\Git\mingw64\share\git\git-for-windows.ico
    • 新建 command –> Open in Bash : G:\huajieli\Git\bin\git-bash.exe

1.4 安装 node.js

  • NODE_PATH –> G:\software exe\node exe\node_modules
  • 在 Path 中添加 G:\software exe\node exe（有node.exe的目录）
